# GerenciadorTarefas 

Projeto desenvolvido durante as aulas do curso técnico em Informática para Internet do [SENAI-MG](https://www.fiemg.com.br/senai/), com o objetivo de criar um gerenciador de tarefas no C#. 

## Funcionalidades  

- **Adicionar Tarefa**
- **Listar Tarefas**
- **Concluir Tarefa**
- **Remover Tarefa**
